Constellation Activity

About this Project

Constellation Activity is an offline planetarium integrated for Sugarizer (a web version of the Sugar Learning Platform). I suggested to the open source leader that VirtualSky, the planetarium provider, to be made available to Sugarizer, and I then volunteered to integrate it and make the controls and GUI more "child-friendly" as Sugarizer is intended for 6-12 years old. This web app help students and children learn more about the constellations, the name of the stars, and the movements of planets, anywhere on earth. Check it out and see! :)
