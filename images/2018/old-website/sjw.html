Space Junk Wars

About this Project

Space Junk Wars is a web game about cleaning space junks that is orbiting the Earth's atmosphere. It was originally made by my team (which happened to be my sisters) for Penang International Science Fair's Scratch Programming Competition. The game won the Champion title, and my team decided to convert it into a web game and deploy it online so others can play it as well.
